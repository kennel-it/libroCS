springBoot

scrivere velocemente dei servizi REST

Spring Boot è una estensione del framework Spring il cui scopo è di mettere facilmente in esecuzione applicazioni web scritte in Java.

Si tratta di un framework cioè di un insieme di componenti preconfezionati che facilita lo sviluppo di software, spesso utilizzando specifiche strategie di sviluppo.

Un framework si differenzia da quella chiamiamo libreria in alcuni aspetti:

  • la modalità con cui si svolgono le operazioni è dettata dal framework e non da chi lo usa
  • esistono una serie di copmportamenti di default
  • chi usa il framwerk può estendere le sue funzionalità ma non modificarlo

Perché Spring Boot

  • riduce di molto il tempo di sviluppo aumentando la produttività
  • evita la scrittura parti di codice ripetitivo e file di configurazione
  • è molto facile accedere ai database e salvare oggetti Java
  • "Opinionated Defaults Configuration" in pratica in base alle dipendenze decide delle configurazioni di default
  • fornisce un server web integrato per sviluppare le applicazioni (tomcat o altri)

Trattandosi di un framework faremo la sua conoscenza direttamente utilizzandolo! Quello che anddremo a realizzare è un sistema per la gestione di una lista di cose da fare.
